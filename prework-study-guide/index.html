Prework Study Guide

HTML

  • Hypertext markup language(HTML) is used to create the structure of a webpage
  • The head element holds metadata not visible to the user
  • The body element contains the content shown to the user
  • HTML elements are case sensitive

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the contents of an element
  • link will link a style sheet to an html sheet
  • A class selector allows to assign CSS rules to specific elements

Git

  • git status checks what branch we are currently on
  • checkout moves the working branch to a new one
  • -b creates a new branch
  • git commands always start with git
